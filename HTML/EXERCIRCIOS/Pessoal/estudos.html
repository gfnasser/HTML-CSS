4) Usando imagens no site

Primeiro você precisa escolher uma imagem, e lembre-se de tomar cuidado com o tamanho da imagem e o direito autoral dela. Escolha imagens que sejam livres para uso ou de sua prorpia criação.
Após escolher as imagens, salve elas na pasta do arquivo HTML, caso elas estejam muito grande, basta redimensionar usando o GIMP.
Após ter feito isso, você vai precisar usar a tag "img", basta escrever essas três letras e apertar enter, que irá aparecer a tag completa, assim: <img src="" alt="">. Interessante é saber o que significa src e alt, resumidamente src é "origem de imagem" e alt é "texto que explica a imagem".
Para usar as imagens que salvamos na pasta, vamos utilizar um atalho para tornar mais prático, que é o Ctrl+Backspace.

Vamos começar carregando uma imagem que está na pasta.

O código fica assim:
<img src="logo1.png" alt="Logo 1">


Agora vamos carregar uma imagem que está na sub-pasta (uma pasta criada dentro de outra).

O código fica assim:
<img src="imagem/logo2.png" alt="Logo 2">


Por último, vamos carregar uma imagem da internet utilizando o link da imagem.

O código fica assim:
<img src="https://styles.redditmedia.com/t5_2tugi/styles/communityIcon_7yzrvmem0wi31.png" alt="Logo internet">


Com isso nós vimos como podemos colocar imagem em nosso site.
Fique sempre atento a dimensão da imagem, tamanho do arquivo e se você tem direito de usa-lá.

5) Colocando uma Favicon no site ou ícone de favoritos

Favicon é o ícone que aprece quando acessamos um site, como o YouTube, Facebook, Google e outros sites. Para exemplificar, observe a imagem abaixo, nela está marcado em vermelho o site que não tem uma favicon e está marcado com verde os sites que possuem favicon.

Você pode tanto criar a sua imagem que será transformada em favicon, quanto buscar em sites na internet que possuem diversos favicon, sites como www.icoarchive.com posuem diversos modelos de favicon, mas lembre-se sempre de baixar a imagem no formato ICO.
Caso você queira criar o seu favicon, você pode usar o site www.favicon.cc.
Também é possivel pegar a imagem(ícone) que você escolheu e tranformar ele em um arquivo ICO, você pode usar o site www.favicon.io. Neste site é possivel tranformar até um texto em favicon, das opções de criação de favicon, essa é a mais Interessante.

Lembre-se sempre de salvar as imagens na pasta do arquivo HTML.

Mas como tonar a imagem em uma favicon?

No inicio do arquivo HTML, dentro de <head></head> ou aréa de cabeças, em cima de <title></title>, você vai digitar a palavra "link" e escolher a opção "link:favicon". Feito isso irá aparecer a tag <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">. Dentro de "favicon.ico" você irá apagar esse texto e apertar Ctrl+BackSpace(atalho rápido), então irá aparecer os ícones que você salvou dentro da pasta do arquivo HTML, agora é só escolher qual favicon você irá querer usar.
Tendo feito isso, a tag fica assim: <link rel="shortcut icon" href="Icons8.ico" type="image/x-icon">

Pronto, agora temos uma favicon.

11) Mídias

Imagem dinâmica

O que é uma imagem dinâmica? É uma imagem que se adapta ao dispositivo que está sendo usado, como por exemplo: computador, celular, tablet, entre outros. Essa imagem irá se adaptar a cada tamanho de tela, mas para isso acontecer nós precisamos ter a mesma imagem em diversos tamanhos, assim era ficará em uma boa resolução em cada dispositivo, evitando que tenha aquela barra lateral para conseguir ver a imagem como um todo.
Mas como fazemos isso? Iremos usar a tag "picture". Dentro desta tag iremos colocar a imagem usando a tag "img", selecionares primeiro a nossa maior imagem. Após ter feito isso, usaremos a tag "source" para adicionar os diversos tamanho da mesma imagem. As imagens devem estar na mesma pasta para facilitar. Lembre-se de que nessa configuração de "source", os arquivos dentro desta tag devem ser colocados em ordem crescente, sendo que o maior arquivo tem que estar dentro da tag "img".

A tag completa fica assim:

    <picture>
    <source media="(max-width: 750px)" srcset="imagem/p.png" type="image/png">
    <source media="(max-width: 1050px)" srcset="imagem/m.png" type="image/png">
    <img src="imagem/g.png" alt="imagem flexivel">
    </picture>

Áudio

Vamos aprender a reproduzir audios em HTML5.

Temos algmas formas de fazer isso, uma delas é usando a tag "audio" e alguns "source". Neste caso nós iremos usar o "source:media:type" visto que iremos usar uma mídia (áudio).
Iremos ver abaixo duas formas de usar o áudio no site, uma mais simples e outras que utilizamos varios formatos do arquivo para que ele seja compativel com o maximo de navegadores. Caso o navegador não sejá capaz de ler as mídias, podemos deixar o link para download.

      <audio src="midias/timetospare-anjone.mp3" autoplay controls></audio>
      <audio preload="metadata" controls autoplay loop>
        <source src="midias/ulcereggaeton-anjone.mp3" type="audio/mpeg">
        <source src="midias/ulcereggaeton-anjone.ogg" type="audio/ogg">
        <source src="midias/ulcereggaeton-anjone.wav" type="audio/wav">
        <p>Caso o seu navegador não reproduza o áudio, <a href="midias/ulcereggaeton-anjone.mp3" download="ulcereggaeton-anjone.mp3" type="audio/mpeg">clique aqui para baixar.</a></p>
      </audio>

Vamos entender os principais tributos da tag "audio" antes de mais nada.

  • O tributo preload indica se o áudio será pré-carrgado ou não e aceita até três valores:
    • metadata, vai carregar as informações sobre o arquivo (tamanho, tempo).
    • none, não vai carregar nada até que o usuario clique no botão play.
    • auto(padrão), vai carregar o arquivo do audio inteirio assim que o site for carregado.
  • O atributo controls vai apresentar o player na tela, caso não sejá colocado dentro da tag "audio" o player não irá aparecer.
  • O atributo autoplay vai iniciar a reprodução do áudio assim que a página for carregada.
  • O atributo loop vai fazer com que o áudio seja repetido eternamente.
Dentro da tag "audio" colocamos varios "source" com formatos diferentes do mesmo áudio.

É importante saber que no Google Chrome não está funcionando o "autoplay".

Vídeo

Inserindo videos hospedados localmente

Para colocar vídeos nós usamos a tag "video". Com essa tag nós conseguimos colocar um video que já baixmamos no site, é importante nos atentarmos para o tamanho do arquivo, ele não pode ser muito grande, senão irá gastar muito plano de dados e o valor da hospedagem do site será muito caro.

É importante termo o video em varios formatos, pois alguns formatos não são suportados pelos navegadores, por isso usamos os formatos mp4, m4v, webm e ogv, com esses formatos conseguimos garantir que em grande parte dos navegadores o video será carregado. Para colocar todos este formatos, nos usamos dentro da tag "video" o "source:src", assim como foi feito ao colocarmos as imagens dinâmicas.

Iremos usar um vídeo hospedado em nosso proprio servidor, veja abaixo como fica o código:

<video controls loop width="500" poster="imagens/tambneio.jpg"> <source src="videos/meus-videos.mp4" type="video/mp4"> <source src="videos/meus-videos-1.webm" type="video/webm"> <source src="videos/meus-videos.ogv" type="video/ogv"> <source src="videos/meus-videos-1.m4v" type="video/m4v"> <p>Seu computador não tem compatibilidade com reprodução de vídeos.</p> </video>

Como pode ver, dentro da tag "video" "controls", "width" e "poster".
Mas o que cada uma faz?

  • Controls: Permite que o usuario dê play e pause no vídeo.
  • Width: Reduz a resolução do video para que ele se encaixe melhor na tela.
  • Poster: Coloca uma tambneio no video.
  • Loop: Faz com que o video sempre reinicie quando acabar.
