Usando links externos

Para para linkar um texto a um site externo, nós envelopamos o texto que queremos linkar e usamos a tag "a", dentro de "href=" nós colocamos o link que queremos que a pessoa seja direcionada ao clicar.
Para o link não sobrepor o seu site, dentro da tag "a", após o link do site, use "target="_blank"".

Você pode acessar o meu repositório público no Github.


E para informar a natureza do link, nós podemos usar "rel=next, prev, author ou external, dentro da tag "a".

Você também pode acessar o meu canal gratuito no Youtube.

Usando links internos

Links internos são links que fazem a ligação entre outras paginas do mesmo servidor.

Para criar uma nova página, basta ir em "explorador" no "VSC", dentro da pasta principal que você já esta digitando o código, e criar um novo "arquivo". Pronto, você agora tem uma nova página dentro do mesmo servidor.
Para fazer as transições entre as paginas nós usamos também a tag "a". Com essa tag nós conseguimos ir e voltar entre as páginas.

Está é a primeira página do meu site, se você quiser pode também visitar a minha segunda página.


No exemplo abaixo foi criado uma nova pasta chamada "noticias" dentro da pasta do "ex010", esssa pasta pode ser criada dentro do "VSC" usando o "explorador". A ideia é acessar uma página que está dentro de outra pasta. Nós conseguimos acessar essa pasta normlamente usando a tag "a", mas na hora de voltarmos para a página principal, vamos precisar adicionar na teg "a" em "href=" os seguintes sinais "../" assim vai tornar possivel retonarmos para a página principal que está em outra pasta.

Links para download

Vamos ver como gerar links para download. Para fazer isso nós continuamos usando a tag "a", mas adicionamos outras infromações dentro da tag, como "download" e "type". Arquivos em PDF não são mais baixados automaticamente pelo navegador Google Chrome, no navegador você é levado a outra página que te da a opção de baixar o arquivo. Já utilizando arquivos em ZIP ou 7z, eles são baixados automaticamente.

Mas como fica a teg quando vamos como colocar um link para download?
Primeiro vamos ver como fica a tag para baixar PDF e depois para baixar ZIP.

1) <a href="Livro/10 - Ligações em toda parte.pdf" download="10 - Ligações em toda parte." type="application/pdf" target="_blank" rel="external">Baixar o livro em pdf.</a>

2) <a href="Livro/10 - Ligações em toda parte.7z" download="10 - Ligações em toda parte.7z" type="application/7z">Livro compactado em zip.</a>
  • Baixar o livro em pdf.
  • Livro compactado em zip.
