Listas Ordenadas

A tag "ol" significa "ordem listas" ou "listas ordenadas", essa tag é usada em conjunto com a tag "li" para criar listas. Pode-se criar listas númericas, alfabeticas ou outras, para fazr isso colocamos dentro da tag "ol" a palavra "type" ai podemos escolher como ficará a nossa lista. Pode-se também escolher como ela irá começar, adicionando também a palavra "starte". Veremos a baixo como fica a tag usando todos estes recusos.

Listas não Ordenadas

A tag "ul" significa "lista desordenada" ou "lista não ordenada", essa tag é usada em conjunto com a tag "li" para criar listas não ordenadas. Pode-se usar "type="cycle", "disc" ou "square", dentri da tag "ul" para alterar o formato da lista não ordenada
