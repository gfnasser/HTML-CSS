Uso da tag "pre" é usado para formatar o texto conforme está o "code", pode ser usado para substituir a necessidade do uso da tag "br" dentro do "code". Dentro da tag "pre" pode-se usar o atalho shift+tab para ajusatar o espaceamento lateral ou usar apenas a tecla tab para aumentar o espaceamento lateral, mas em ambas as situações é necessário primeiro selecionar o texto.

Citações

Sem quebra de linha

Para fazer citações que não possuem quebra de linha, nós usamos a tag "q".

Como diria o pai de um amigo: o computador é um burro muito rápido.

Citações com quebra de linha

Para fazer citações com quebra de linhas nós usamos a tag "blockquote", que faz a citação com um espaceamento lateral. Dentro desta tag podemos usar "cite" e colocarmos o link original da citação.

Segundo Alexandre Dumas, no seu livro O Conde de Monte Cristo:

A vida é uma tempestade (...) Um dia você está tomando sol e no dia seguinte o mar te lança contra as rochas. O que faz de você um homem é o que você faz quando a tempestade vem.

Abreviações

Para mostrar o siginifacado das abreviações nos podemos usar a tag "abbr" e colocar o significado de cada abreviação, com isso é só a pessoa passar o mouse em cima da abreviação que irá aparcer o significado.
