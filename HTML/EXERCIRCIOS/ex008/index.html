Principais formatações

Uso de atalho:
Procure usar o atalho Ctrl+Shift+P (Quebra libra com abreviação ou Wrap with Abbreviation), este atalho vai te ajudar a ser mais rapido no uso das tags.

Negrito ou destaque

Primeiramente, exitem duas maneiras de colocar uma palavra em negrito, de uma maneira não semântica e de uma maneira semântica.

A "não semântica" é simplesmente pela forma, como o exemplo abaixo mostra:
Nessa frase, temos um termo em negrito usando a tag "B".

Já a "semântica" usa o significado, conforme o exemplo abaixo:
Nesta frase, temos um termo em destaque usando a tag "STRONG".

Itálico ou ênfase

Igual as palavras em negrito, as palavras em itálico existem formas semânticas e não semânticas.

Nesta frase, temos um termo em itálico usando a tag "I"(não semântica).

Nesta frase, temos um termo em destaque usando a tag "EM"(semântica).

Uso da tag "B" e "I".

Visualmente não exite nenhum difença entre as tags semântica e não semântica de itálico e negrito, mas apesar disso, a tag "B" e "I" ela fazem parte do HTML4. Agora nós estamos no HTML5, que é focado na semântica, então, evite usar as tags sem semântica, porque ela podem derrepente parem de ser usadas, dito isto, procure usar as tags semânticas, ou seja, as tags "STRONG" e "EM".

Texto marcado

Podemos criar também um texto marcado usando a tag "MARK".
E você pode alterar a cor do marcador de texto suando a tag "STYLE" dentro da tag inicial "MARK".
Ficando a tag assim:
O ceú é <mark style="background-color: rgb(7, 157, 245);"> azul </mark>.

Mas e se eu quiser colocar todas as tags "MARK" em uma cor que não seja em amarelo, que é a padrão, vou ter que selecionar uma por uma?
Uma opção é você ir no cabeçalho do código, em baixo de "TITLE", e colocar a tag "STYLE", depois coloque a tag "MARK" e adicione um seletor ( { } ), então dentro do seletor você vai colocar "background-color" e escolher qual cor você quer.
Vejá abaixo como fica:

<title>Exemplo</title>
<style>
mark
{
background-color: red;
}
</style>

Lembrando, essa tag tem que ser feita no cabeçalho.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste paragráfo.
Usamos a tag "BIG" para letra grande e "SMALL" para letra pequena. Mas a tag "BIG" está em vermelha, o que nos diz que está tag esta obsoleta e logo mais deixará de funcionar. Então, não se deve usar a tag "BIG" para ter letras grandes.

Texto deletado

Podemos marcar um texto como excluido(deletado) para indicar que ele deve ser lido, mas não considerado.
Usando a tag "DEL".

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adiconado depois.
Usando a tag "INS".

Existe também o sublinhado usando a tag "U" que é a forma não semântica.

Texto sobreescrito

Para inserir coisas do tipo "x20+3".
Para isso usamos a tag "SUP". SUP é em cima.
A tag fica assim: x<sup>20</sup>+3.

Texto subscrito

Para inserir coisas do tipo "H2O"
Para isso usamos a tag "SUB". SUB é em baixo.
A tag fica assim: H<sub>2</sub>O.
