1) Parágrafo e quebra de linha.

Você pode escrever um parágrafo de qualquer jeito, basta colocar tudo no meio do par de tags <p> e </p>.
Sendo <p> o inicio do parágrafo.
Sendo </p> o final do parágrafo.

Se você precisar quebrar o texto em algum lugar específico você precisa usar a tag <br>.

Ex:
Maria foi ao mercador,<br>
mas esqueceu a carteira.

Dessa maneira descrita acima vai ficar mais facil de entender como funciona o uso da tag <br>.

2) Tags iniciais.

Algumas das tags que já foram apresentadas.Todas elas são usadas em minúsculo.

p= Parágrafo
br= Quebra de linha
title= titulo que aparece na parte superior do anvegador
h1= Tema ou titulo do que vai ser escrito (resumidamente)
hr= Linha horizontal
(&)lt; e (&)gt;= Serve para mostrar as tags usadas nos textos.
< !-- -->= Serve para colocar um comentário, esse comentário não aparece no site.
&#x= É usados para adicionar emoji, você usa essa tag mais o código do emoji e finaliza com ponto e vírgula (;).

4) Usando imagens no site

Primeiro você precisa escolher uma imagem, e lembre-se de tomar cuidado com o tamanho da imagem e o direito autoral dela. Escolha imagens que sejam livres para uso ou de sua prorpia criação.
Após escolher as imagens, salve elas na pasta do arquivo HTML, caso elas estejam muito grande, basta redimensionar usando o GIMP.
Após ter feito isso, você vai precisar usar a tag "img", basta escrever essas três letras e apertar enter, que irá aparecer a tag completa, assim: <img src="" alt="">. Interessante é saber o que significa src e alt, resumidamente src é "origem de imagem" e alt é "texto que explica a imagem".
Para usar as imagens que salvamos na pasta, vamos utilizar um atalho para tornar mais prático, que é o Ctrl+Backspace.

Vamos começar carregando uma imagem que está na pasta.

O código fica assim: <img src="logo1.png" alt="Logo 1">.


Agora vamos carregar uma imagem que está na sub-pasta (uma pasta criada dentro de outra).

O código fica assim: <img src="imagem/logo2.png" alt="Logo 2">.


Por último, vamos carregar uma imagem da internet utilizando o link da imagem.

O código fica assim: <img src="https://styles.redditmedia.com/t5_2tugi/styles/communityIcon_7yzrvmem0wi31.png" alt="Logo internet">.


Com isso nós vimos como podemos colocar imagem em nosso site.
Fique sempre atento a dimensão da imagem, tamanho do arquivo e se você tem direito de usa-lá.
